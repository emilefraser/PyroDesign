File upload with Gears

Select a single file, and upload it using Gears. This sample allows files 20MB or smaller, but gears will handle any size file. The limit can be changed to any amount by modifiying UploadServlet.

When an upload completes, you will see a message on the console printed by the server displaying the file name and the temp file it was written to. This sample uses an "X-Filename" header to pass the name of the selected file.
